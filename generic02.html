Introduction to Data Types and Formats

This unit examined diverse data types and formats and how these affect how data is stored with particular emphasis on data file formats. It further analysed different data formats such as structure, quasi, semi and unstructured data formats, as well as the implementation of APIs (Application Programming Interfaces) to facilitate seamless parsing of data between various storage systems and processes.
